Scrapspace

Workspace for your Scrapbox projects

Scrapspace is an unofficial desktop client for Scrapbox. Please contact @shoya140 instead of NOTA Inc. if you have any problems regarding this application. Release Notes v0.1.0 (2018.09.09) - Published the first alpha version - Supporting WindowsOS v0.0.8 (2018.09.01) - cmd + t : Open a new tab - cmd + Number : Go to the tab v0.0.7 (2018.08.31) - shift + cmd + n : Open a new window v0.0.6 (2018.08.30) - Displaying a progress bar for the cache loading - cmd + o : Go to any URL - cmd + n : Create a new page v0.0.5 (2018.08.29) - cmd + p : Cros-search - shift + cmd + r: Storing chaches for the cross-search v0.0.1 (2018.08.27) - Started this project
